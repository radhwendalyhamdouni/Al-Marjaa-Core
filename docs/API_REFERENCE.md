## نظرة عامة

لغة المرجع هي لغة برمجة عربية متكاملة مصممة خصيصاً للمتحدثين بالعربية.

### المميزات الرئيسية

- **بناء جمل عربي** - كتابة الكود باللغة العربية بالكامل
- **JIT Compiler** - أداء عالي مع الترجمة الفورية
- **نظام أنواع قوي** - أنواع ثابتة مع استنتاج تلقائي
- **مكتبة قياسية شاملة** - دوال جاهزة للاستخدام

## 🚀 الميزات المتقدمة (الإصدار 4.0)

### Register-based VM (أسرع 2-3x)

لغة المرجع تستخدم الآن آلة افتراضية مبنية على السجلات بدلاً من المكدس، مما يوفر أداءً أفضل:

\`\`\`
// المقارنة:
// Stack VM: Push a, Push b, Add, Pop result (4 تعليمات)
// Register VM: Add r0, r1, r2 (تعليمة واحدة)
\`\`\`

**المميزات:**
- 256 سجل متاح
- تعليمات مدمجة (Add dest, src1, src2)
- Type tracking للتحسين التلقائي
- Inline caching للمتغيرات العامة
- Cache locality محسّن

### FFI (Foreign Function Interface)

استدعاء دوال من مكتبات C/Rust الخارجية:

\`\`\`mrj
// تحميل مكتبة خارجية
استيراد ffi

// تحميل مكتبة
ffi.حمّل("libc", "/usr/lib/libc.so.6")

// استدعاء دالة C
متغير نتيجة = ffi.استدعِ("libc", "printf", ["مرحباً بالعالم"])

// استخدام الدوال المدمجة
متغير ذاكرة = ffi.تخصيص(1024)     // تخصيص ذاكرة
ffi.نسخ_ذاكرة(هدف، مصدر، 1024)    // نسخ ذاكرة
ffi.تحرير(ذاكرة)                   // تحرير الذاكرة
\`\`\`

**أنواع FFI المدعومة:**
| النوع | الوصف |
|-------|-------|
| عدد_32 | عدد صحيح 32-bit |
| عدد_64 | عدد صحيح 64-bit |
| عشري_32 | عدد عشري 32-bit |
| عشري_64 | عدد عشري 64-bit |
| مؤشر | مؤشر للذاكرة |
| نص | نص منتهي بـ null |
| دالة | دالة callback |

### SIMD Operations (تسريع 4-8x)

عمليات المتجهات للعمليات الحسابية المتوازية:

\`\`\`mrj
// جمع مصفوفتين باستخدام SIMD
متغير أ = [1.0، 2.0، 3.0، 4.0]
متغير ب = [5.0، 6.0، 7.0، 8.0]
متغير نتيجة = simd.جمع(أ، ب)  // [6.0، 8.0، 10.0، 12.0]

// عمليات SIMD المتاحة
simd.جمع(أ، ب)        // جمع عنصر ب عنصر
simd.طرح(أ، ب)        // طرح
simd.ضرب(أ، ب)        // ضرب
simd.قسمة(أ، ب)       // قسمة
simd.جذر(أ)           // جذر تربيعي لكل عنصر
simd.مطلق(أ)          // قيمة مطلقة
simd.أقصى(أ، ب)       // أقصى قيمة
simd.أدنى(أ، ب)       // أدنى قيمة
simd.ناتج_نقطة(أ، ب)  // dot product

// عمليات المنطق
simd.و(أ، ب)          // AND
simd.أو(أ، ب)         // OR
simd.ليس(أ)           // NOT
\`\`\`

**مثال عملي - معالجة صور:**

\`\`\`mrj
// تفتيح صورة باستخدام SIMD
دالة تفتيح_صورة(بكسلات، عامل):
    // معالجة 4 بكسلات في المرة الواحدة
    متجر نتيجة = []
    لكل مجموعة في simd.مجموعات(بكسلات، 4):
        متغير مضاعف = simd.ضرب(مجموعة، عامل)
        نتيجة.أضف(مضاعف)
    أرجع نتيجة
\`\`\`

### JIT Compiler مع SIMD

الـ JIT Compiler يستخدم SIMD تلقائياً للعمليات الحسابية:

\`\`\`mrj
// هذا الكود يُحسّن تلقائياً باستخدام SIMD
دالة مجموع_مربعات(قائمة):
    متغير مجموع = 0
    لكل عنصر في قائمة:
        مجموع += عنصر * عنصر  // ← يُترجم لـ SIMD
    أرجع مجموع
\`\`\`

**مستويات التحسين JIT:**
| المستوى | التنفيذات | التحسينات |
|---------|----------|----------|
| Tier 0 | 0 | Interpreter Baseline |
| Tier 1 | 50 | Baseline JIT |
| Tier 2 | 200 | Optimizing JIT |
| Tier 3 | 1000 | SIMD Optimizations |
| Tier 4 | 5000 | Tracing JIT |
