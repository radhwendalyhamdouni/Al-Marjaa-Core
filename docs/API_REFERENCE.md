## نظرة عامة

لغة المرجع هي لغة برمجة عربية متكاملة مصممة خصيصاً للمتحدثين بالعربية.

### المميزات الرئيسية

- **بناء جمل عربي** - كتابة الكود باللغة العربية بالكامل
- **JIT Compiler** - أداء عالي مع الترجمة الفورية
- **نظام أنواع قوي** - أنواع ثابتة مع استنتاج تلقائي
- **مكتبة قياسية شاملة** - دوال جاهزة للاستخدام

## البداية السريعة

### التثبيت

\`\`\`bash
git clone https://github.com/radhwendalyhamdouni/Al-Marjaa-Core.git
cd Al-Marjaa-Core
cargo build --release
./target/release/almarjaa
\`\`\`

### أول برنامج

\`\`\`mrj
// ملف: مرحبا.mrj
طباعة("مرحبا بالعالم!")
\`\`\`

## أنواع البيانات

### الأنواع الأساسية

| النوع | الوصف | مثال |
|-------|-------|------|
| رقم | أعداد صحيحة وعشرية | 42، 3.14 |
| نص | سلسلة نصية | "مرحبا" |
| منطقي | قيمة منطقية | صحيح، خطأ |
| لا_شيء | قيمة فارغة | لا_شيء |

### الأنواع المركبة

| النوع | الوصف | مثال |
|-------|-------|------|
| قائمة | مجموعة مرتبة | [1، 2، 3] |
| قاموس | أزواج مفتاح-قيمة | {اسم: "أحمد"} |
| دالة | كود قابل للاستدعاء | دالة(س) => س * 2 |
| صنف | قالب لكائنات | صنف شخص {...} |

\`\`\`mrj
// دالة أساسية
دالة جمع(أ، ب):

متغير مضاعف = دالة(س) => س * 2

## الفئات

\`\`\`mrj
صنف شخص:
    متغير اسم = ""
    متغير عمر = 0
    
    دالة جديد(اسم، عمر):
        هذا.اسم = اسم
        هذا.عمر = عمر
    
    دالة معلومات():
        أرجع "الاسم: " + هذا.اسم
\`\`\`

## التحكم في التدفق

### الشروط

\`\`\`mrj
إذا س > 10:
    طباعة("كبير")
وإلا إذا س == 10:
    طباعة("متوسط")
وإلا:
    طباعة("صغير")
\`\`\`

### الحلقات

\`\`\`mrj
// while
بينما س > 0:
    س = س - 1

// for
لـ س من 1 إلى 10:
    طباعة(س)

// foreach
لكل عنصر في قائمة:
    طباعة(عنصر)
\`\`\`

## مرجع الدوال

### الإدخال والإخراج

| الدالة | الوصف |
|--------|-------|
| طباعة(...) | طباعة للشاشة |
| إدخال(رسالة) | قراءة من المستخدم |

### النصوص

| الدالة | الوصف |
|--------|-------|
| نص.طول() | طول النص |
| نص.حرف_كبير() | تحويل لكبير |
| نص.حرف_صغير() | تحويل لصغير |
| نص.جزء(من، إلى) | اقتطاع جزء |
| نص.استبدل(قديم، جديد) | استبدال |

### الأرقام

| الدالة | الوصف |
|--------|-------|
| abs(س) | القيمة المطلقة |
| round(س) | التقريب |
| sqrt(س) | الجذر التربيعي |
| pow(أ، ب) | الأس |

## المكتبة القياسية

### التشفير

\`\`\`mrj
تشفير.md5("نص")
تشفير.sha256("نص")
تشفير.aes.شفر("نص"، "مفتاح")
\`\`\`

### الوقت

\`\`\`mrj
متغير الآن = وقت.الآن()
الآن.سنة()
الآن.شهر()
الآن.يوم()
الآن.تنسيق("YYYY-MM-DD")
\`\`\`

### JSON

\`\`\`mrj
متغير json = json.مشفر(قاموس)
متعرض بيانات = json.مفكوك(json)
\`\`\`
