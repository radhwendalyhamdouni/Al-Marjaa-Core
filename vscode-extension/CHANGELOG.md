# Changelog

## [1.0.0] - 2024-03-21

### الميزات الجديدة

#### تلوين الكود (Syntax Highlighting)
- تلوين كامل للكلمات المفتاحية العربية
- دعم الأرقام العربية (٠١٢٣) والإنجليزية
- تلوين النصوص والتعليقات
- دعم الأرقام السداسية عشر والثنائية

#### خادم اللغة (Language Server)
- الإكمال التلقائي الذكي
- عرض التوثيق عند التحويم
- مساعدة توقيعات الدوال
- اكتشاف الأخطاء في الوقت الحقيقي
- البحث عن التعريفات والمراجع
- عرض رموز المستند

#### التصحيح (Debugger)
- دعم نقاط التوقف
- التنقل خطوة بخطوة
- فحص المتغيرات
- تقييم التعبيرات

#### التقطيعات البرمجية (Snippets)
- أكثر من 50 تقطيعة برمجية
- دعم كامل للكلمات العربية
- قوالب للدوال والفئات والحلقات

#### أوامر الإضافة
- `marjaa.run` - تشغيل البرنامج
- `marjaa.debug` - تصحيح البرنامج
- `marjaa.compile` - تجميع البرنامج
- `marjaa.format` - تنسيق الملف
- `marjaa.startServer` - بدء خادم اللغة
- `marjaa.stopServer` - إيقاف خادم اللغة
- `marjaa.restartServer` - إعادة تشغيل خادم اللغة

#### الإعدادات
- `marjaa.executablePath` - مسار المفسر
- `marjaa.enableLanguageServer` - تفعيل خادم اللغة
- `marjaa.format.enable` - تفعيل التنسيق
- `marjaa.format.indentSize` - حجم المسافة البادئة
- `marjaa.diagnostics.enable` - تفعيل التشخيص
- `marjaa.completion.enable` - تفعيل الإكمال
- `marjaa.hover.enable` - تفعيل التحويم

### الإصلاحات
- إصلاح مشاكل التوافق مع UTF-8
- تحسين أداء التحليل
